# Getting Started

To build a Typst project you can follow the instructions in the [Typst repository](https://github.com/typst/typst?tab=readme-ov-file#installation).

## Command to preview the pdf
Watches source files and recompiles on changes (saves).
```bash
typst watch main.typ
```
> Best for developing mode

## To only generate the pdf
```bash
typst compile main.typ
```
